## How to use
To run this project yoy need:
  1. Clone this repo:

    `git clone https://github.com/OSpyrydonov/MyBike-landing.git`

  2. Open this repo on your IDE:

    `code MyBike-landing`

  4. Istall depedencies:

    `npm i`
    
or
    
    `npm install`

  5. Start the project:

    `npm start`
